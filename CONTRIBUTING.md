# Contributing to UrbanWear

Thank you for your interest in contributing to UrbanWear!

## Development Workflow
1. Fork and clone the repository.
2. Set up virtual environment and install dependencies.
3. Create a feature branch.
4. Make sure your changes follow Django best practices.
5. Commit and open a Pull Request.

## Testing Requirements
Before submitting any changes, please ensure that:
1. All existing tests pass.
2. New unit tests are written for any new features or model helper methods.
3. You run the complete test suite locally:
   ```bash
   python storeproject/manage.py test
   ```
